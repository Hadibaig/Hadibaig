I don’t just build websites — I develop and maintain production-ready WordPress plugins as standalone products with long-term ownership, scalability, and real-world problem-solving focus.

My plugin portfolio includes:

* **Bulk Category Manager**
  Advanced bulk taxonomy and category management system for WordPress, designed to simplify large-scale content organization and editing workflows.

  Lightweight security plugin for customizing and protecting WordPress login URLs, improving site security and reducing brute-force attack risks.
